## 📍 RECORD VISIT (2 Minutes)

1. Tap **"New Visit"** tab
2. Tap **"Get GPS Location"** (FIRST!)
3. Wait for ✓ green checkmark
4. Fill all fields:
   - ZBM Name
   - Outlet Name
   - Contact: 09XXXXXXXX
   - Agent Code
   - Town
   - Cluster
   - Market
   - Float Amount (ZMW)
   - Has Float (Yes/No)
5. Tap **"Save Visit"**
6. Done!
