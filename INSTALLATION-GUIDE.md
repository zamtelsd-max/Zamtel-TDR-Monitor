## 📍 RECORDING YOUR FIRST VISIT

### Step 1: Enable Location

Before using the app, **enable location services**:

**Android:**
- Settings → Location → Turn ON
- Settings → Apps → Chrome → Permissions → Location → Allow

**iPhone:**
- Settings → Privacy → Location Services → Turn ON
- Settings → Safari → Location → Allow

---

### Step 2: Record Visit

1. **Tap "New Visit"** tab at the bottom
2. **Tap "Get GPS Location"** button (pink button)
   - Allow location permission if asked
   - Wait for green checkmark
3. **Fill in all fields**:
   - ZBM Name
   - Outlet Name
   - Contact Number (Format: 0977XXXXXX or 0966XXXXXX)
   - Agent Code
   - Town
   - Cluster Name
   - Market Name
   - Float Amount (in Kwacha)
   - Has Float (Yes/No)
4. **Tap "Save Visit"** (green button)
5. **Done!** Visit is saved

## ❓ COMMON PROBLEMS & SOLUTIONS

### ❌ Problem: "GPS Location Not Working"

**Solutions:**
1. Check location is turned ON in phone settings
2. Allow location permission in browser
3. Go outside or near a window for better GPS signal
4. Try closing and reopening the app
5. Restart your phone

---

### ❌ Problem: "App Not Installing"

**Android Solutions:**
- Use Chrome browser (not Firefox)
- Update Chrome to latest version
- Clear Chrome cache
- Check you have storage space

**iPhone Solutions:**
- Use Safari browser (not Chrome!)
- Update iOS to latest version
- Clear Safari cache
- Check you have storage space

---

### ❌ Problem: "Form Not Submitting"

**Solutions:**
1. Make sure GPS location was captured (green checkmark)
2. Check phone number format (09XXXXXXXX or 07XXXXXXXX)
3. Fill in ALL required fields
4. Check internet connection
5. Try again after few seconds

---

### ❌ Problem: "Can't See My Visits"

**Solutions:**
1. Tap "Visits" tab at bottom
2. Check filter settings at top
3. Make sure you're logged in with correct name
4. Scroll down to see all visits

---

### ❌ Problem: "Forgot to Get GPS Location"

**Solution:**
- You MUST tap "Get GPS Location" BEFORE filling the form
- If you forgot, clear the form and start again
- GPS must show green checkmark before submitting
